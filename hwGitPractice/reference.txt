clone: creates a remote repository in the terminal, this repository is the same thing as the url needed to execute this command

push: loads all changes to github

pull: updates any changes made on github

commit: saves changes in the remote repository

reset: undoes certain commits

status: shows the repository

diff: shows changes made to the wanted file

init: initializes a repository

add: adds or updates a file to the staging area

log: lists changes made

checkout: undoes a commit
